📁 EHB-AI-Dev-Phase-1.zip

🧩 Yeh ZIP file EHB AI Developer Agent ka Phase 1 setup provide karti hai.

📂 Folder Structure:
- frontend/: Next.js frontend with TailwindCSS
- backend/: Express.js API with AI endpoint
- models/: MongoDB Mongoose models
- config/: MongoDB config file
- admin/: Future admin panel (empty abhi)
- public/: Static files placeholder
- README.txt: Setup guide

⚙️ Setup Instructions (Replit ya VS Code):
1. Replit pe ZIP upload karo, unzip karo.
2. .env file banao aur MONGO_URI set karo.
3. Terminal se `cd backend && npm install && node index.js` chalao.
4. Ek aur terminal me `cd frontend && npm install && npm run dev` chalao.

🧠 AI Agent: Abhi dummy response de raha hai, lekin prompt send karna test kar sakte ho via POST /api/ai/ask.

✅ Tools Used: Next.js, TailwindCSS, Express, MongoDB, JWT (aane wale phase me)

📌 Har folder well-commented aur future phases ke liye tayar hai.
