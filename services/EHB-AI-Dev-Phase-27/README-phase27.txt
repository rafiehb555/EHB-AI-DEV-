📘 Phase 27: Service Subscription + Payment History

✅ Components:
- Frontend: UI for managing active subscriptions and viewing history
- Backend: Routes + controller for managing plans, subscribing users, saving payments
- Models: Subscription & PaymentHistory schemas
- Admin: Control panel for subscription plan creation and user overview

🛠 Setup Instructions:
- Paste folders into your project under respective paths
- Make sure MongoDB + JWT is configured
- Use Axios or Fetch to test API endpoints

📁 Folder Naming:
- frontend-phase27/
- backend-phase27/
- models-phase27/
- admin-phase27/

🧠 Purpose:
Allow users to subscribe to services, track their payment history, and give admins full control on backend.
